# LINEAR ALGEBRA

* 선형대수?
    * 공간을 설명하는 데 편리한 용어나 개념 제공
    * 선형 대수가 다루는 대상은 선형(직선, 평면 등 곧은 것)
* 벡터/행렬/행렬식의 의미
    * 벡터 : 화살표 또는 공간 안의 점
    * 행렬 : 공간에서 공간으로의 직교 사상
    * 행렬식 : 위의 사상에 따른 부피 확대율
* 벡터의 덧셈/정수배
    * 덧셈 : 화살표의 이어 붙임
    * 정수배 : 길이 늘임/줄임
* 선형공간
    * 벡터 공간
    * 덧셈/정수배가 정의된 세계
* 기저
    * 기준이 되는 벡터
